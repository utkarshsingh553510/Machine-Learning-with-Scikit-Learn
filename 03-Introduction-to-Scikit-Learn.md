# Scikit-Learn

## Introduction

**Scikit-Learn** Python ki ek bahut popular **Machine Learning Library** hai.

Is library ki help se hum easily Machine Learning Models bana sakte hain.

Scikit-Learn me pehle se hi bahut saare Algorithms available hote hain.

Isliye hume algorithms ko scratch se likhne ki zarurat nahi padti.


Ex-> Let say hume Ek Cake banana hai to 

X-> Ingradient

y-> Taste 

Recipe-> Model 

Practice -> Training 

Taste-> Testing 

Agar Taste Kharab Hai to ise hum Aur Behtar Bana Skate Hai i.e Improving Model 

---

## Features of Scikit-Learn

- Easy to Learn
- Open Source Library
- Python Based
- Fast & Efficient
- Large Community Support
- Multiple Machine Learning Algorithms Available
- Data Preprocessing Support
- Model Evaluation Tools
- Cross Validation Support

# Installing Scikit-Learn

Scikit-Learn ko install karne ke liye `pip` package manager ka use kiya jata hai.

## Installation

```bash
pip install scikit-learn
```

Agar Jupyter Notebook use kar rahe ho to:

```python
!pip install scikit-learn
```

Installation check karne ke liye:

```python
import sklearn

print(sklearn.__version__)
```

Agar version print ho jaye to installation successfully complete ho chuki hai.

# Main Modules of Scikit-Learn

## 1. sklearn.model_selection

Use:

- Train-Test Split
- Cross Validation
- Hyperparameter Tuning

Example:

```python
from sklearn.model_selection import train_test_split
```

---

## 2. sklearn.preprocessing

Use:

- Label Encoding
- One Hot Encoding
- Standardization
- Normalization

Example:

```python
from sklearn.preprocessing import StandardScaler
```

---

## 3. sklearn.linear_model

Use:

- Linear Regression
- Logistic Regression

Example:

```python
from sklearn.linear_model import LinearRegression
```

---

## 4. sklearn.tree

Use:

- Decision Tree Classifier
- Decision Tree Regressor

Example:

```python
from sklearn.tree import DecisionTreeClassifier
```

---

## 5. sklearn.neighbors

Use:

- K-Nearest Neighbors (KNN)

Example:

```python
from sklearn.neighbors import KNeighborsClassifier
```

---

## 6. sklearn.cluster

Use:

- K-Means Clustering

Example:

```python
from sklearn.cluster import KMeans
```

---

## 7. sklearn.metrics

Use:

Model Performance Check karne ke liye.

Metrics:

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

Example:

```python
from sklearn.metrics import accuracy_score
```
