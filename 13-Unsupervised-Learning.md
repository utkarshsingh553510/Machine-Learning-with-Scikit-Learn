##  Unsupervised Learning

Unsupervised Learning में हम Machine को **Unlabeled Data** देते हैं।

Machine खुद Data में Common Patterns ढूंढती है और Similar चीज़ों को अलग-अलग **Groups (Clusters)** में Divide करती है।

### Example

Let Say एक Basket में तीन प्रकार के Fruits हैं:

- Apple
- Orange
- Banana

लेकिन उन Fruits पर कोई **Tag** या **Label** नहीं लगा है।

अब हमें उन्हें Group करना है।

तो हम:

- Same Size वाले Fruits को एक Group में रख देंगे।
- Same Shape वाले Fruits को एक Group में रख देंगे।
- Same Color वाले Fruits को एक Group में रख देंगे।

यही Unsupervised Learning कहलाती है।
